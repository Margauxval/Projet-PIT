# Projet-PIT — Apprenti Bashicien

Jeu éducatif terminal pour apprendre les commandes UNIX, version française.

## Installation rapide (Debian/Ubuntu)
1. Décompresse l'archive.
2. Rends les scripts exécutables :
   chmod +x start generate_seed.sh countdown.sh check_bomb_status.sh modules/*.sh
3. Lance le jeu :
   ./start

Chaque module est interactif : tape les commandes demandées dans le prompt du module.
